# Mishmash_Hackathon

A Flutter based Mobile Application addressing the problems of CoronaVirus

## Setup

```
git clone https://github.com/mayurnile/gocorona.git
cd gocorona
flutter packages get
flutter run(or just press F5 in VS Code)

```
